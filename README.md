## Install packages

```shell
composer install
```

```shell
npm install
```

## Copy environment configuration

```shell
cp .env.example .env
```

## Run migrations

```shell
php artisan migrate
```

## Initial Database Seeding

```shell
php artisan db:seed
```

## Run development server

```shell
npm run dev
```

##

```shell
php artisan serve
```

## Build for production

```shell
npm run build
```
